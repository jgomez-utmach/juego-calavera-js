  ** Cuando el tamaño de pantalla cambia debemos recargar la página para volver a calcular el tamaño del canvas
  ** Para solucionar este problema usaremos un addEventListener para volverlo responsivo

  - Declaramos un addEventListener para escuchar el evento 'resize'
  - Separamos la lógica de la función draw en:
    -> función para calcular el tamaño del canvas
    -> función para dibujar en el canvas
  - Declaramos como variables globales a:
    -> let canvasSize;
    -> let elementsSize;
  - Añadimos función que calcula el tamaño del canvas al evento 'resize'

Metodos usados:
  .addEventListener('resize', función);
